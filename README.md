# limemath
This Rust crate is a simple math crate that has a new invented operator called **Termi**, which the value is **100**. This crate has only 3 files:

## Files
* `operators/`:
    * `mod.rs`: This is when Termi will born
    * `termi.rs`: This is when Termi is created

* `lib.rs`: The entry point that the lib is born
